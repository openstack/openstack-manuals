Ocata Installation Guides

The following guides support basic installation of core and optional OpenStack services on variants of Red Hat, SUSE, and Ubuntu distributions. A minimal deployment necessary to launch a simple instance (VM) requires following the core content. Optional content augments a minimal deployment with additional services.

openSUSE and SUSE Linux Enterprise
Install core and optional OpenStack services on an openSUSE or SUSE Linux Enterprise system, using a manual installation method.
Red Hat Enterprise Linux 7 and CentOS 7
Install core and optional OpenStack services on a Red Hat Enterprise Linux 7 or CentOS 7 system, using a manual installation method.
Ubuntu
Install core and optional OpenStack services on an Ubuntu system, using a manual installation method.
